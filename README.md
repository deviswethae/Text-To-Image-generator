# Text-to-Image-Generator

# Description

This project is a Text-to-Image Generator that converts textual descriptions into visually realistic images using AI-powered image generation models. It provides an intuitive interface where users can enter prompts and instantly generate high-quality images.

The application demonstrates the integration of AI APIs with a modern web stack to deliver a seamless and interactive creative experience.

# Features

Text Prompt Input: Users can enter descriptive text to generate images.

AI Image Generation: Generates high-quality images based on user prompts.

Download Images: Users can download generated images.

History Preview: Displays previously generated images.

Responsive UI: Works smoothly across desktop and mobile devices.

Fast Processing: Optimized API calls for quick image generation.

# Technologies Used

React.js – Frontend library for building interactive user interfaces.

Node.js – JavaScript runtime for backend processing.

Express.js – Backend framework for API handling.

AI Image API – Used to generate images from text prompts.

Axios – For handling API requests.

CSS / Tailwind / Chakra UI (depending on what you used) – Styling and responsive design.

# Installation
1. Clone the repository
git clone https://github.com/deviswethae/Text-To-Image-generator.git
2. Navigate to the project folder
cd Text-To-image-generator
3. Install backend dependencies
cd backend
npm install
4. Install frontend dependencies
cd frontend
npm install
5. Start Backend Server
cd backend
npm start
6. Start Frontend
cd frontend
npm start

The application will run at:

👉 http://localhost:3000

# Usage

Enter a descriptive prompt (e.g., “A futuristic city at sunset with flying cars”).

Click Generate Image.

View the generated image.

Download or save the image.
